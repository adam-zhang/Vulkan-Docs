VkImageType(3)
==============

Name
----
VkImageType - Specifies the type of an image object.

C Specification
---------------

include::../enums/VkImageType.txt[]

Constants
---------

ename:VK_IMAGE_TYPE_1D::
    One-dimensional image type.

ename:VK_IMAGE_TYPE_2D::
    Two-dimensional image type.

ename:VK_IMAGE_TYPE_3D::
    Three-dimensional image type.

Description
-----------

The constants of this enumeration are used to specify the type of an image object created using
the flink:vkCreateImage command.

--------
